## 🚀 What is cia?

cia is a desktop app for managing background agents in your Elixir app. It helps you start, stop, and monitor agents without digging through logs or terminal commands. Use it when you want a simple way to keep background work under control from one place.

## 🧰 What you need

Before you install cia on Windows, make sure you have:

- A Windows 10 or Windows 11 PC
- An internet connection
- Permission to install and run apps on your computer
- An Elixir app if you plan to use cia with your own project

If you are not sure whether your PC can run it, a modern Windows laptop or desktop is usually enough.

## 🖥️ Install on Windows

1. Find the downloaded cia file
2. If it is a ZIP file, right-click it and choose Extract All

## ❓ Troubleshooting

### The app does not open
- Make sure you downloaded the full file
- Extract the ZIP folder first if needed
- Try running the app as administrator

### I cannot find my Elixir app
- Open cia settings
- Check the project path
- Make sure the folder contains your app files

### An agent will not start
- Confirm the agent is set up in your app
- Check the agent name
- Restart cia and try again

### Windows blocks the file
- Right-click the file and open Properties
- If you see an unblock option, select it
- Try downloading the file again from the GitHub page
